# Data Science Project

A beginner-friendly Data Science and Machine Learning project built using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

This project performs:
- Data Cleaning
- Data Analysis
- Data Visualization
- Feature Encoding
- Model Training
- Model Evaluation

# Features

- Load dataset using Pandas
- Handle missing values
- Remove duplicate records
- Generate statistical summaries
- Create histograms and correlation heatmaps
- Encode categorical data
- Train Machine Learning models
- Compare model performance

# Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
  
# Machine Learning Models Used

1. Linear Regression
2. Random Forest Regressor
